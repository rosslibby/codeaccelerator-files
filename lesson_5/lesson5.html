Lesson 5: More CSS

Margins & Padding

Margins

We can space out HTML elements using margins. A margin is a space around an element.

      

____________________________
|                           |
|  Element with no margin   |
|___________________________|


        Now use the CSS property "margin" to add a margin around the element

        .my-class {
          margin: 30px;
        }



        _________________________
        |                        |
        |  Element with margin   |
        |________________________|

Padding

Padding adds size to an element instead of just putting space around it.

      
        button with no padding


        Now use the CSS property "padding" to add padding (the left number deals with top/bottom padding, the right number deals with left/right padding)

        .my-class {
          padding: 8px 12px;
        }


        button with padding

That's the exact same code in each button, but one is obviously more readable than the other.

Borders

Borders are just outlines on elements. Here, I'll show you.

      
        
This is a div with no border



        Now use the CSS property "border" to add a border. You need to specify the following three things in this property's value:

        size (in pixels)
        style (dotted/dashed/solid/double/groove/ridge/inset/outset)
        color

        We'll be going with 1 pixel (1px), solid, and black (#000000)

        .my-class {
          border: 1px solid #000000;
        }

        
This is a div with a border

Width & Height

See how the div with the border fills up the whole page? Unless we specify that the div should not fill the entire page, it always will.

We can get around this by using the CSS width and height properties.

      
        
This is a div with no width or height


        Now use the CSS property "width" to set the width. You can set it in pixels (px) or as a percentage (%). We'll set it to 50% for now.

        .my-class {
          width: 50%;
        }

        
This is a div with a width but no height


        Ok so that's pretty cool, but it's still VERY short. Let's add a height using the CSS property "height". This time we'll use pixels instead of a percentage. Let's do 100px. And let's keep the width we set previously.

        .my-class {
          width: 50%;
          height: 100px;
        }

        
This is a div with a width and a height
