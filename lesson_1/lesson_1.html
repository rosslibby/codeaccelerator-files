HTML

HTML stands for HyperText Markup Language.

HTML is essentially the thing that makes text and photos show up on websites.

HTML is markup - it is NOT code. Writing HTML is like using Microsoft Word, but instead of clicking buttons you have to use something called tags.

Tags

An HTML tag is very simple: it is a letter or word between two angle braces.

An angle brace is what we call the greater than (>) and less than (<) symbols. An HTML tag looks like this:

<html>

I am going to show you a few other tags. By the end of this lesson, you will understand how HTML works and be able to write and format text in a website.

Tag examples

  • <b> makes text bold
  • <i> makes text italic
