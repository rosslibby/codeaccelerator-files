Lesson 4: An intro to CSS

What is CSS?

CSS stands for Cascading Style Sheets

While HTML is used to describe the content of a webpage, CSS is used to present the content. In other words it makes things look good.

CSS Classes

CSS can interact with the HTML that you write through something called a class. A class is a structure that contains various properties and can be applied to a range of elements. This means that I can make multiple parts of the HTML look the same without writing code over and over.

A CSS class name must always start with a dot. Example:

      .my-classname

A CSS class will always have braces in which its properties will reside. 

      
        .my-classname {
        }

CSS properties will go between the braces. Properties are rules that anything within that specific CSS class will follow.

      
        .text--red {
          color: #ff0000;
        }

How to mix CSS with HTML

Just writing CSS itself doesn't do anything. We must also include the CSS file in the HTML. We do this using linked files like we learned about in lesson 2.

      
        <html>
          <head>
            <link rel="stylesheet" href="style.css" />
          </head>
        </html>

This still doesn't do anything. We have to actually make sure that our HTML knows that the CSS classes exist, otherwise they won't take affect.

      
        <body>
          <div class="text--red">
            
This is some exciting text

          </div>
        </body>
